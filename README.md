# Cashew Leaf Disease Classification using ResNet50

A CNN-based image classification system using **ResNet50** to classify diseases in cashew leaves.

## Technologies

- Python
- TensorFlow
- Keras
- ResNet50

## Dataset

- Healthy
- Anthracnose
- Leaf Miner
- Red Rust

## Model Performance

- Validation Accuracy: **±89%**
